excel表格刷题系统
=========================
![](https://img.shields.io/badge/excel--to--test-v1.0.1-blue.svg)

介绍 
------------
将excel表格题目导出为题库

用法
------------
* 下载文件
```
  git pull git@github.com:scarqin/excel-to--test-library.git

```
* 编辑表格
表头必须要有
    * 序号
    * 题型
    * 题干
    * 选项A、选项B...[最多到F]
    * 答案
